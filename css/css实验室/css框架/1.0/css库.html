表格样式库

表格标签

标签描述
(table)用以包装格式化的数据
（thead)容纳表格标题行
（tbody)容纳表格主要内容行
（tr）容纳行内的一组单元格（td)(th)
(td)默认的表格单元格
(th)用过列头（thead）内

表格样式选项

名称 类 描述
默认 None 无样式，仅仅有列和行
基本 .table 行与行之间以水平线相隔
有边线 .table-bordered 表格外围均有外边框
间隔背景 .table-striped 奇数行背景设为浅灰色
紧凑 .table-condensed 竖直方向padding缩减一半，从8px变为4px，所有的 td 和 th 元素都受影响

按钮样式库

按钮样式

按钮类 class=“”描述
.btn标准灰色按钮
.btn .btn-primary视觉吸引力强 用于标识一组安妮中最主要的那个动作
.btn .btn-info可用于替换默认样式
.btn .btn-success表示操作成功或动作正确
.btn .btn-warning表示动作应谨慎
.btn .btn-danger表示危险或有潜在威胁的动作
.btn .btn-inverse做为补充的深灰色按钮，与具体行为或动作无关。
